# UselessBinding (FL0057)

## Cause

An identifier is binded to itself e.g. `let x = x`.

## Rationale

Pointless statement likely to be an error.

## How To Fix

Remove the binding.

## Rule Settings

    {
        "uselessBinding": {
            "enabled": true
        }
    }
