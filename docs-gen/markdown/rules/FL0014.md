# RedundantNewKeyword (FL0014)

## Cause

Using `new` to instantiate a type which does not implement `IDisposable`.

## Rationale

`new` is redundant for insantiating types which do not implement `IDisposable`.

## How To Fix

Remove `new` keyword.

## Rule Settings

	[lang=javascript]
    {
        "redundantNewKeyword": {
            "enabled": false
        }
    }
